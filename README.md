# 🚀 前端静态页面实战合集 

## 🌟 核心项目展示

### 1. 🎮 王者荣耀官网仿站 
- **技术亮点**：
  - 使用 **Flexbox** 结合 **绝对定位** 完美复现官网复杂的英雄卡片列表。
  - **CSS 动画 (@keyframes)**：实现了高度自适应的下拉菜单 (`dropmenu`) 与平滑的显隐过渡。
  - **视觉交互**：运用 `transform: scale` 与 `transition` 实现了图片缩放与遮罩交互。

### 2. 🎵 网易云音乐首页 
- **技术亮点**：
  - **标准流与混合布局**：严格遵守标准文档流，结合 `flex` 与 `float` 处理网易云经典的侧边栏与主内容区布局。
  - **csst图片**：运用 **CSS Sprite (精灵图)** 

### 3. 📐 Grid 布局深度练习 
- **技术亮点**：
  - 深度使用 **CSS Grid** 完成非对称布局设计。
  - 掌握了 `grid-template-areas` 等高级属性，摆脱了传统的浮动布局困扰。

---

## 🛠️ 技术栈
- **HTML5**：语义化标签运用。
- **CSS3**：Flexbox、Grid、Keyframes 动画、媒体查询。
  - **高级布局**: Grid (网格布局)、Flexbox (弹性盒)。
  - **进阶特性**: 线性渐变 (Linear Gradient)、伪元素 (Pseudo-elements)、CSS Transitions/Animations。
